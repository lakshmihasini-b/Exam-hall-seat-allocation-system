# Exam Hall Seat Allocation System (C++)
## Description
The Exam Hall Seat Allocation System is a simple C++ program that automatically assigns exam hall numbers and seat numbers to students based on user-defined seating capacity per hall. The system ensures that each student receives a unique seat allocation and prevents duplicate roll number entries.
## Features
- User-defined number of seats per exam hall
- Automatic allocation of hall and seat numbers
- Duplicate roll number validation
- Supports multiple exam halls
- Saves seating arrangement to a file
- Displays allocation in a clear tabular format
## Technologies Used
- Programming Language: C++
- Concepts Used: Arrays , Loops , Conditional statements , File handling
